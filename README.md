## About This Repository

This repository contains my hands-on work done as part of the **VSD FPGA Internship screening**.
The focus is on understanding how a digital design moves from **high-level Verilog code to a synthesized gate-level implementation**.

Through these modules, I explored:

* Writing and simulating RTL designs
* Building and using testbenches
* Observing signal behavior through waveforms
* Converting RTL into netlists using synthesis tools
* Understanding how standard cell libraries influence design
